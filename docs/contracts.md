## Vault Interface (key functions)

```solidity
// Shield tokens into the privacy pool
function shield(address token, uint256 amount, uint256 commitment0, uint256 commitment1) external;

// Shield native ETH (auto-wraps to WETH internally)
function shieldEth(uint256 commitment0, uint256 commitment1) external payable;

// Private send — spend a note, create new note(s) for recipient
function send(
    address token,
    uint256[24] calldata proof,
    uint256 root,
    uint256 nullifierHash,
    uint256 recipientCommitment,
    uint256 changeCommitment,
    bytes32 txHash
) external;

// Unshield — withdraw to a public address
function unshield(
    address token,
    uint256[24] calldata proof,
    uint256 root,
    uint256 nullifierHash,
    uint256 amount,
    address to,
    uint256 changeCommitment,
    bytes32 txHash
) external;
```

## Supported Tokens

Any ERC-20 registered by governance can be shielded. Currently enabled:

- **USDC** (Base)
- **WETH** (Base)

2. Click "Contract" → "Read Contract" to inspect state
3. Click "Contract" → "Code" to view verified Solidity source
